# 逼歌

- B 哥/逼哥/南京李志 歌曲下载
- 歌单、专辑封面已进行本地化，歌曲资源从网络下载
- 下载采用分块下载，支持断点续传

## icon 生成

```
flutter pub run icons_launcher:create
```

## 说明

- 使用 flutter 开发
- 歌单地址 [list-v2.js](https://testingcf.jsdelivr.net/gh/nj-lizhi/song@master/audio/list-v2.js)
